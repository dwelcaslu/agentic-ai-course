# 🤖 Agentic AI — Course Notes & Projects

This repository contains notes, code implementations, and experiments based on the **Agentic AI** course by Andrew Ng, available exclusively on DeepLearning.AI.

---

## 📚 About the Course

The **Agentic AI** course focuses on building intelligent systems that go beyond simple prompt-response interactions. Instead of generating single outputs, these systems:

- Plan multi-step workflows  
- Take actions using tools  
- Iterate and improve results over time  

Agentic AI represents a shift from **reactive AI** to **autonomous, goal-driven systems** capable of executing complex tasks.

---

## 🚀 What You’ll Learn

- 🧠 **Agentic Design Patterns**
  - Reflection (self-improvement loops)
  - Tool Use (APIs, databases, code execution)
  - Planning (task decomposition)
  - Multi-Agent Systems (collaborative agents)

- 🔗 **Tool Integration**
  - Connect AI to external systems (APIs, web search, databases)

- 📊 **Evaluation & Optimization**
  - Metrics, error analysis, and production readiness

- 🏗️ **Production-Ready Systems**
  - Build scalable and reliable agentic workflows

---

## 🧩 Course Structure

The course is organized into 5 main modules:

1. **Introduction to Agentic AI**
2. **Reflection Design Pattern**
3. **Tool Use Design Pattern**
4. **Practical Tips for Building Agentic Systems**
5. **Planning & Multi-Agent Systems**

🎯 **Capstone Project:**  
Build a **Research Agent** capable of gathering information, analyzing it, and generating structured outputs autonomously.
